## Questions

### Question 1: Bob's Nature
What makes Bob an "agentic" AI assistant?

A) Bob only answers questions  
B) Bob can autonomously use tools and make decisions to accomplish tasks  
C) Bob requires explicit instructions for every step  
D) Bob can only read files  

---

### Question 2: Mode Selection
Which Bob mode should you use to understand existing code without making changes?

A) Code mode  
B) Plan mode  
C) Ask mode  
D) Orchestrator mode  

---

### Question 3: Code Mode Capabilities
What can Bob do in Code mode?

A) Only read files  
B) Read and modify files, execute commands, and use Git  
C) Only answer questions  
D) Only create new files  

---

### Question 4: Orchestrator Mode
When should you use Orchestrator mode?

A) For simple single-file changes  
B) For asking questions only  
C) For large multi-file projects and complex workflows  
D) For reading documentation  

---

### Question 5: Effective Communication
Which is the BEST way to request help from Bob?

A) "Make it better"  
B) "Add user authentication with JWT tokens to the API"  
C) "Do something"  
D) "Fix stuff"  

---

## Answers

<details>
<summary>Click to reveal answers</summary>

### Answer 1: B
Bob is agentic because it can autonomously use tools, make decisions, and work through complex tasks step-by-step without constant guidance.

### Answer 2: C
Ask mode is perfect for understanding existing code, getting explanations, and code reviews without making any changes.

### Answer 3: B
Code mode allows Bob to read and modify files, execute commands, use Git, and perform various development tasks (but not browser tools or MCP servers).

### Answer 4: C
Orchestrator mode is designed for large multi-file projects, full-stack applications, and complex workflows that require coordination across multiple tasks.

### Answer 5: B
Clear, specific requests like "Add user authentication with JWT tokens to the API" give Bob a concrete goal while allowing it to figure out the implementation details autonomously.
