## Questions

### Question 1: Command Palette
What is the keyboard shortcut to open the Command Palette?

A) `Cmd+P` (Mac) / `Ctrl+P` (Windows/Linux)  
B) `Cmd+Shift+P` (Mac) / `Ctrl+Shift+P` (Windows/Linux)  
C) `Cmd+K` (Mac) / `Ctrl+K` (Windows/Linux)  
D) `Cmd+B` (Mac) / `Ctrl+B` (Windows/Linux)  

---

### Question 2: Multi-Cursor Editing
How do you add a cursor to the next occurrence of the selected text?

A) `Cmd+D` (Mac) / `Ctrl+D` (Windows/Linux)  
B) `Cmd+Shift+L` (Mac) / `Ctrl+Shift+L` (Windows/Linux)  
C) `Alt+Click`  
D) `Cmd+U` (Mac) / `Ctrl+U` (Windows/Linux)  

---

### Question 3: Search and Replace
Which keyboard shortcut opens the Find and Replace dialog?

A) `Cmd+F` (Mac) / `Ctrl+F` (Windows/Linux)  
B) `Cmd+H` (Mac) / `Ctrl+H` (Windows/Linux)  
C) `Cmd+Shift+F` (Mac) / `Ctrl+Shift+F` (Windows/Linux)  
D) `Cmd+R` (Mac) / `Ctrl+R` (Windows/Linux)  

---

### Question 4: Code Navigation
What does "Go to Definition" (F12) do?

A) Opens the file browser  
B) Jumps to where a function or variable is defined  
C) Shows all references to a symbol  
D) Renames a symbol  

---

### Question 5: Multi-Cursor
Which of these is NOT a way to create multiple cursors?

A) `Alt+Click` at different positions  
B) `Cmd+Alt+Up/Down` (Mac) / `Ctrl+Alt+Up/Down` (Windows/Linux)  
C) `Cmd+D` (Mac) / `Ctrl+D` (Windows/Linux) on selected text  
D) `Cmd+S` (Mac) / `Ctrl+S` (Windows/Linux)  

---

## Answers

<details>
<summary>Click to reveal answers</summary>

### Answer 1: B
`Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux) opens the Command Palette, which gives access to all VS Code commands.

### Answer 2: A
`Cmd+D` (Mac) or `Ctrl+D` (Windows/Linux) adds a cursor to the next occurrence of the currently selected text.

### Answer 3: B
`Cmd+H` (Mac) or `Ctrl+H` (Windows/Linux) opens Find and Replace. `Cmd+F` / `Ctrl+F` opens just Find.

### Answer 4: B
"Go to Definition" (F12) jumps to where a function, variable, or class is defined in your codebase.

### Answer 5: D
`Cmd+S` / `Ctrl+S` saves the file - it doesn't create multiple cursors. All other options are valid ways to create multiple cursors.
