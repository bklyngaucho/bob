### Question 2: Productivity Best Practice
Which of the following is considered a best practice for productivity?

A) Using the mouse for all operations  
B) Learning and using keyboard shortcuts

D) Never using extensions

### Question 3: Quick Fix
What keyboard shortcut triggers Quick Fix suggestions?

A) `Cmd+.` (Mac) / `Ctrl+.` (Windows/Linux)  
B) `Cmd+/` (Mac) / `Ctrl+/` (Windows/Linux)  
C) `F2`  
D) `Alt+Enter`  

---

### Question 4: Workflow Optimization
What is the benefit of using the Command Palette frequently?

A) It makes VS Code run faster  
B) It provides quick access to all commands without memorizing shortcuts  
C) It automatically saves your files  
D) It closes unused tabs  

---

## Answers

<details>
<summary>Click to reveal answers</summary>

### Answer 1: B
Zen Mode (`Cmd+K Z` / `Ctrl+K Z`) hides all UI elements (sidebar, status bar, etc.) to provide a distraction-free coding environment.

### Answer 2: B
Learning and using keyboard shortcuts is a key productivity best practice. It significantly speeds up your workflow compared to using the mouse.

### Answer 3: A
`Cmd+.` (Mac) or `Ctrl+.` (Windows/Linux) triggers Quick Fix suggestions, which can automatically fix common issues or provide refactoring options.

### Answer 4: B
The Command Palette provides quick access to all VS Code commands without needing to memorize every keyboard shortcut, making it a powerful productivity tool.
