# Module 6.1: Keyboard Shortcuts Cheatsheet

## Quick Reference Guide

This cheatsheet covers the most essential VS Code keyboard shortcuts for maximum productivity.

## File Operations

```
Cmd+N / Ctrl+N                    New file
Cmd+O / Ctrl+O                    Open file
Cmd+S / Ctrl+S                    Save
Cmd+Shift+S / Ctrl+Shift+S        Save as
Cmd+W / Ctrl+W                    Close file
Cmd+K W / Ctrl+K W                Close all files
Cmd+Shift+T / Ctrl+Shift+T        Reopen closed file
```

## Navigation

```
Cmd+P / Ctrl+P                    Quick Open (files)
Cmd+Shift+O / Ctrl+Shift+O        Go to Symbol
Cmd+T / Ctrl+T                    Go to Symbol in Workspace
Cmd+G / Ctrl+G                    Go to Line
F12                               Go to Definition
Cmd+- / Ctrl+-                    Go Back
Cmd+Shift+- / Ctrl+Shift+-        Go Forward
```

## Editing

```
Cmd+X / Ctrl+X                    Cut line
Cmd+C / Ctrl+C                    Copy line
Cmd+V / Ctrl+V                    Paste
Cmd+Z / Ctrl+Z                    Undo
Cmd+Shift+Z / Ctrl+Shift+Z        Redo
Cmd+/ / Ctrl+/                    Toggle comment
Option+Up/Down / Alt+Up/Down      Move line up/down
Shift+Option+Up/Down              Copy line up/down
Cmd+Shift+K / Ctrl+Shift+K        Delete line
Cmd+Enter / Ctrl+Enter            Insert line below
Cmd+Shift+Enter                   Insert line above
```

## Multi-Cursor

```
Option+Click / Alt+Click          Add cursor
Cmd+Option+Up/Down                Add cursor above/below
Cmd+D / Ctrl+D                    Select next occurrence
Cmd+Shift+L / Ctrl+Shift+L        Select all occurrences
Cmd+K Cmd+D / Ctrl+K Ctrl+D       Skip occurrence
Shift+Option+Drag / Shift+Alt+Drag Column selection
```

## Search and Replace

```
Cmd+F / Ctrl+F                    Find
Cmd+H / Ctrl+H                    Replace
Cmd+Shift+F / Ctrl+Shift+F        Find in files
Cmd+Shift+H / Ctrl+Shift+H        Replace in files
Cmd+G / F3                        Find next
Cmd+Shift+G / Shift+F3            Find previous
Option+Enter / Alt+Enter          Select all matches
```

## View Management

```
Cmd+B / Ctrl+B                    Toggle sidebar
Cmd+J / Ctrl+J                    Toggle panel
Cmd+` / Ctrl+`                    Toggle terminal
Cmd+\ / Ctrl+\                    Split editor
Cmd+1,2,3 / Ctrl+1,2,3            Focus editor group
Cmd+K Z / Ctrl+K Z                Zen mode
```

## Terminal

```
Cmd+` / Ctrl+`                    Toggle terminal
Ctrl+Shift+`                      New terminal
Cmd+\ / Ctrl+Shift+5              Split terminal
Cmd+K / Ctrl+K                    Clear terminal
```

## Code Folding

```
Cmd+Option+[ / Ctrl+Shift+[       Fold
Cmd+Option+] / Ctrl+Shift+]       Unfold
Cmd+K Cmd+0 / Ctrl+K Ctrl+0       Fold all
Cmd+K Cmd+J / Ctrl+K Ctrl+J       Unfold all
```

## Formatting

```
Shift+Option+F / Shift+Alt+F      Format document
Cmd+K Cmd+F / Ctrl+K Ctrl+F       Format selection
```

## IntelliSense

```
Ctrl+Space                        Trigger suggestions
Cmd+. / Ctrl+.                    Quick fix
Cmd+Shift+Space / Ctrl+Shift+Space Parameter hints
```

## Debugging

```
F5                                Start/Continue
F9                                Toggle breakpoint
F10                               Step over
F11                               Step into
Shift+F11                         Step out
Shift+F5                          Stop
Cmd+Shift+F5 / Ctrl+Shift+F5      Restart
```

## Git

```
Cmd+Shift+G / Ctrl+Shift+G        Source Control view
Cmd+Enter / Ctrl+Enter            Commit (in SCM)
```

## Command Palette

```
Cmd+Shift+P / Ctrl+Shift+P        Command Palette
F1                                Command Palette
```

## Extensions

```
Cmd+Shift+X / Ctrl+Shift+X        Extensions view
```

## Settings

```
Cmd+, / Ctrl+,                    Settings
Cmd+K Cmd+S / Ctrl+K Ctrl+K       Keyboard shortcuts
```

## Tab Management

```
Cmd+W / Ctrl+W                    Close tab
Cmd+K W / Ctrl+K W                Close all tabs
Cmd+K Cmd+W / Ctrl+K Ctrl+W       Close other tabs
Cmd+Option+Left/Right             Previous/Next tab
Cmd+1,2,3... / Ctrl+1,2,3...      Go to tab number
```

## Platform Differences

### macOS
- `Cmd` = Command key
- `Option` = Alt/Option key
- `Ctrl` = Control key

### Windows/Linux
- `Ctrl` = Control key
- `Alt` = Alt key
- `Shift` = Shift key

## Tips for Learning

1. **Start with 5 shortcuts** - Master before adding more
2. **Print this cheatsheet** - Keep visible while coding
3. **Use Command Palette** - Shows shortcuts for commands
4. **Practice daily** - Build muscle memory
5. **Customize wisely** - Only change conflicting shortcuts

## Key Takeaways

- Keyboard shortcuts dramatically improve speed
- Learn essential shortcuts first
- Command Palette is most important
- Practice builds muscle memory
- Customize for your workflow
- Use cheatsheet while learning
