# Module 4.4: Keyboard Shortcuts

## Why Keyboard Shortcuts Matter

Keyboard shortcuts dramatically improve productivity by:
- Reducing mouse usage
- Speeding up common tasks
- Maintaining coding flow
- Reducing context switching

**Goal:** Learn the most impactful shortcuts first, then expand gradually.

## Essential Shortcuts (Learn These First)

### File Operations
```
Cmd+N / Ctrl+N - New file
Cmd+O / Ctrl+O - Open file
Cmd+S / Ctrl+S - Save file
Cmd+Shift+S / Ctrl+Shift+S - Save as
Cmd+W / Ctrl+W - Close file
Cmd+K W / Ctrl+K W - Close all files
```

### Navigation
```
Cmd+P / Ctrl+P - Quick Open (files)
Cmd+Shift+O / Ctrl+Shift+O - Go to Symbol
Cmd+G / Ctrl+G - Go to Line
F12 - Go to Definition
Cmd+- / Ctrl+- - Go Back
```

### Editing
```
Cmd+X / Ctrl+X - Cut line (no selection)
Cmd+C / Ctrl+C - Copy line (no selection)
Cmd+V / Ctrl+V - Paste
Cmd+Z / Ctrl+Z - Undo
Cmd+Shift+Z / Ctrl+Shift+Z - Redo
Cmd+/ / Ctrl+/ - Toggle comment
```

### Search
```
Cmd+F / Ctrl+F - Find
Cmd+H / Ctrl+H - Replace
Cmd+Shift+F / Ctrl+Shift+F - Find in files
Cmd+D / Ctrl+D - Add selection to next find match
```

### View
```
Cmd+B / Ctrl+B - Toggle sidebar
Cmd+J / Ctrl+J - Toggle panel
Cmd+` / Ctrl+` - Toggle terminal
Cmd+\ / Ctrl+\ - Split editor
```

## Command Palette
```
Cmd+Shift+P / Ctrl+Shift+P - Command Palette
F1 - Command Palette (alternative)
```

**Most important shortcut!** Access any command from here.

## Multi-Cursor Editing

### Basic Multi-Cursor
```
Option+Click / Alt+Click - Add cursor
Cmd+Option+Up/Down / Ctrl+Alt+Up/Down - Add cursor above/below
Cmd+D / Ctrl+D - Select next occurrence
Cmd+Shift+L / Ctrl+Shift+L - Select all occurrences
```

### Column Selection
```
Shift+Option+Drag / Shift+Alt+Drag - Column select
Cmd+Shift+Option+Arrow / Ctrl+Shift+Alt+Arrow - Column select (keyboard)
```

## Line Operations

### Moving Lines
```
Option+Up/Down / Alt+Up/Down - Move line up/down
Shift+Option+Up/Down / Shift+Alt+Up/Down - Copy line up/down
```

### Deleting Lines
```
Cmd+Shift+K / Ctrl+Shift+K - Delete line
Cmd+Backspace / Ctrl+Shift+Backspace - Delete to line start
```

### Inserting Lines
```
Cmd+Enter / Ctrl+Enter - Insert line below
Cmd+Shift+Enter / Ctrl+Shift+Enter - Insert line above
```

## Code Folding
```
Cmd+Option+[ / Ctrl+Shift+[ - Fold
Cmd+Option+] / Ctrl+Shift+] - Unfold
Cmd+K Cmd+0 / Ctrl+K Ctrl+0 - Fold all
Cmd+K Cmd+J / Ctrl+K Ctrl+J - Unfold all
```

## Editor Management

### Split Editor
```
Cmd+\ / Ctrl+\ - Split editor
Cmd+1,2,3 / Ctrl+1,2,3 - Focus editor group
Cmd+K Left/Right / Ctrl+K Left/Right - Move editor
```

### Tab Management
```
Cmd+W / Ctrl+W - Close tab
Cmd+K W / Ctrl+K W - Close all tabs
Cmd+K Cmd+W / Ctrl+K Ctrl+W - Close other tabs
Cmd+Shift+T / Ctrl+Shift+T - Reopen closed tab
```

### Tab Navigation
```
Cmd+Option+Left/Right / Ctrl+PageUp/PageDown - Previous/Next tab
Cmd+1,2,3... / Ctrl+1,2,3... - Go to tab number
```

## Terminal Shortcuts
```
Cmd+` / Ctrl+` - Toggle terminal
Ctrl+Shift+` - New terminal
Cmd+\ / Ctrl+Shift+5 - Split terminal
Cmd+K / Ctrl+K - Clear terminal
```

## Git Shortcuts
```
Cmd+Shift+G / Ctrl+Shift+G - Source Control view
Cmd+K Cmd+C / Ctrl+K Ctrl+C - Git commit
```

## Formatting
```
Shift+Option+F / Shift+Alt+F - Format document
Cmd+K Cmd+F / Ctrl+K Ctrl+F - Format selection
```

## IntelliSense
```
Ctrl+Space - Trigger suggestions
Cmd+. / Ctrl+. - Quick fix
Shift+Option+F12 / Shift+Alt+F12 - Peek references
```

## Zen Mode
```
Cmd+K Z / Ctrl+K Z - Toggle Zen mode
Esc Esc - Exit Zen mode
```

## Chord Shortcuts

Chord shortcuts require two key combinations in sequence:

```
Cmd+K Cmd+S / Ctrl+K Ctrl+S - Keyboard shortcuts
Cmd+K Cmd+T / Ctrl+K Ctrl+T - Select theme
Cmd+K Cmd+F / Ctrl+K Ctrl+F - Format selection
Cmd+K Cmd+0 / Ctrl+K Ctrl+0 - Fold all
```

**How to use:**
1. Press first combination (e.g., `Cmd+K`)
2. Release keys
3. Press second combination (e.g., `Cmd+S`)

## Customizing Shortcuts

### View All Shortcuts
```
Cmd+K Cmd+S / Ctrl+K Ctrl+S
```

### Change Shortcut
1. Open Keyboard Shortcuts
2. Search for command
3. Click pencil icon
4. Press new key combination
5. Press Enter

### Remove Shortcut
1. Right-click shortcut
2. Select "Remove Keybinding"

### Add Shortcut
1. Search for command
2. Click `+` icon
3. Press key combination
4. Press Enter

## Shortcut Conflicts

### Resolving Conflicts

**When shortcuts conflict:**
1. VS Code shows warning
2. Choose which to keep
3. Or assign different shortcut

**Check for conflicts:**
```
Keyboard Shortcuts → Search → "@source:user"
```

## Tips for Learning

### Tip 1: Learn Gradually
- Master 5 shortcuts per week
- Use them consistently
- Add more when comfortable

### Tip 2: Print Cheat Sheet
- Keep visible while coding
- Reference frequently
- Remove when memorized

### Tip 3: Use Command Palette
- Shows shortcuts for commands
- Learn shortcuts naturally
- Discover new commands

### Tip 4: Practice Daily
- Use shortcuts intentionally
- Avoid mouse for learned shortcuts
- Build muscle memory

### Tip 5: Customize Wisely
- Only change conflicting shortcuts
- Keep standard shortcuts when possible
- Document custom shortcuts

## Key Takeaways

- Keyboard shortcuts boost productivity
- Learn core shortcuts first
- Command Palette is most important
- Multi-cursor editing is powerful
- Customize shortcuts for your workflow
- Practice builds muscle memory
- Learn gradually, not all at once
- Use cheat sheets while learning
