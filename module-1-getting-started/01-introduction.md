# Module 1.1: Introduction to VS Code

## What is Visual Studio Code?

Visual Studio Code (VS Code) is a free, open-source code editor developed by Microsoft. It's:

- **Lightweight yet powerful**: Fast startup and performance
- **Cross-platform**: Works on Windows, macOS, and Linux
- **Extensible**: Thousands of extensions available
- **Built for developers**: Optimized for coding, debugging, and version control

## Why Use VS Code?

### Key Benefits

1. **IntelliSense**: Smart code completion and suggestions
2. **Built-in Git**: Version control integration
3. **Debugging**: Powerful debugging tools built-in
4. **Extensions**: Customize for any language or framework
5. **Integrated Terminal**: Run commands without leaving the editor
6. **Free and Open Source**: No cost, community-driven

## VS Code vs Other Editors

| Feature | VS Code | Traditional Text Editors | Full IDEs |
|---------|---------|-------------------------|-----------|
| Speed | ⚡ Fast | ⚡ Very Fast | 🐌 Slower |

| Resource Usage | 💚 Light | 💚 Very Light | 🔴 Heavy |

## First Launch

When you first open VS Code, you'll see:

1. **Welcome Screen**: Quick access to recent files and tutorials
2. **Activity Bar** (left side): Main navigation icons
3. **Editor Area** (center): Where you write code
4. **Status Bar** (bottom): Information about your workspace

## Your First Steps

### 1. Open a Folder
```
File → Open Folder... (or Cmd+O on Mac, Ctrl+O on Windows)
```

### 2. Create a New File
```
File → New File (or Cmd+N on Mac, Ctrl+N on Windows)
```

### 3. Save Your Work
```
File → Save (or Cmd+S on Mac, Ctrl+S on Windows)
```

## Understanding the Philosophy

VS Code follows these principles:

- **Code-centric**: Optimized for writing and editing code
- **Keyboard-first**: Most actions have keyboard shortcuts
- **Extensible**: Add features as you need them
- **Workspace-aware**: Understands your project structure

## Common Use Cases

VS Code excels at:

- ✅ Web development (HTML, CSS, JavaScript, TypeScript)
- ✅ Python development
- ✅ Git version control
- ✅ Markdown documentation
- ✅ JSON/YAML configuration files
- ✅ Multi-language projects
- ✅ Remote development (SSH, containers)

## Practice Exercise

**Exercise 1.1: Your First File**

1. Open VS Code
2. Create a new file (Cmd+N / Ctrl+N)
3. Type: `Hello, VS Code!`
4. Save the file as `hello.txt` (Cmd+S / Ctrl+S)
5. Close the file
6. Reopen it using File → Open Recent

**Congratulations!** You've completed your first VS Code task.
